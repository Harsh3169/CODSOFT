# Task 2: Tic-Tac-Toe AI

This is a basic Tic-Tac-Toe game with a simple AI opponent. It uses Python and Tkinter for a graphical user interface.

## Features
- Play as "X" against the AI (which plays as "O")
- Random AI move generation
- Detects win, loss, and draw conditions
- "Play Again" button to restart the game

## Technologies Used
- Python 3
- Tkinter (built-in GUI library)

## How to Run
1. Make sure Python is installed.
2. Run the file with:
   ```bash
   python tictactoe.py
